What file naming does it expect?
  • MM.DD Practice.csv - team-wide practice (e.g. 03.10 Practice.csv)
  • MM.DD Practice Setter.csv - per-setter practice (e.g. 03.10 Practice Gabbi.csv)
  • MM.DD Exhibition Opponent.csv - team exhibition (e.g. 03.28 Exhibition Cleveland State.csv)
  • MM.DD Opponent Setter.csv - per-setter exhibition (e.g. 03.28 Cleveland State Janelle.csv)
